##Aplikasi To-Do-List

##Deskripsi
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian

##Fitur
- Tambah Tugas
- Tandai Tugas Selesai
- Hapus Tugas

##Struktur Folder
index.php - halaman utama

## Cara Menjalankan
1. Salin folder ke "htdocs/"
2. Jalankan XAMPP dan buka di "http://localhost/todolist"
